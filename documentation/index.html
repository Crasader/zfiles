Installations

  1. Download the file which you have purchased
  2. Upload Zip File in your server
  3. Extract the Zip file that you uploaded
  4. Create a Database using hosting database wizard
  5. Now Go to your Browser and Check http://www.yourdomain.com
  6. Give your Database Host Name (Ex: localhost), Database User Name, Password ( That’s given by your Domain & Hosting Provider), Database Name (That’s Name you have already created in number 4) & click Next button.

 

Then Enter your Admin Username, Email address and Password. Save this information because this information will be open your new account and its needed to log in this system. Now Click Install button

4. Click "Install" and that's it :)
